# Image Processing Studio 🎨🖼️

A MATLAB-based graphical interface for image processing and analysis.

## 🚀 Features
- Load and display images
- Convert to grayscale
- Rotate image
- Histogram visualization
- Dark/Light mode interface
- Pixel selection for histogram

## 🛠 Technologies
- MATLAB
- Image Processing Toolbox

## 📂 Project Structure
- main_interface.m
- image_processing_functions.m
- /images
